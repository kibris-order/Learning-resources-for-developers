

### 9. **Find All Triplets with Zero Sum**
- **Problem**: Write a program to find all unique triplets in the array that sum to zero.
- **Input**: An array of integers.
- **Output**: All triplets where the sum is zero.

**Example**:
- Input: `[-1, 0, 1, 2, -1, -4]`
- Output: `[[-1, 0, 1], [-1, -1, 2]]`
