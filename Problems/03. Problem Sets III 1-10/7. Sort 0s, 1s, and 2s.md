
### 7. **Sort 0s, 1s, and 2s**
- **Problem**: Given an array consisting only of 0s, 1s, and 2s, sort the array without using any sorting algorithm (use loops and conditional logic).
- **Input**: An array of 0s, 1s, and 2s.
- **Output**: The sorted array.

**Example**:
- Input: `[2, 0, 2, 1, 1, 0]`
- Output: `[0, 0, 1, 1, 2, 2]`
