
### 6. **Check if Array is Sorted**
- **Problem**: Write a program to check if an array is sorted in non-decreasing order using loops.
- **Input**: An array of integers.
- **Output**: Print "Yes" if the array is sorted, otherwise "No".

**Example**:
- Input: `[1, 2, 3, 4, 5]`
- Output: `Yes`

- Input: `[5, 3, 8, 1]`
- Output: `No`
