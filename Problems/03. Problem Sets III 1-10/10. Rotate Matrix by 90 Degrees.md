

### 10. **Rotate Matrix by 90 Degrees**
- **Problem**: Write a program to rotate an NxN matrix by 90 degrees clockwise.
- **Input**: A square matrix `A` of size `N x N`.
- **Output**: The matrix rotated by 90 degrees.

**Example**:
- Input:  
  `A = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]`
- Output:  
  `[[7, 4, 1], [8, 5, 2], [9, 6, 3]]`
