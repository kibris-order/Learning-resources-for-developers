
### 5. **Multiplication Table**
- **Problem**: Write a program to print the multiplication table of a number `n`.
- **Input**: An integer `n`.
- **Output**: Print the multiplication table for `n` up to 10.

**Example**:
- Input: `3`
- Output:
  ```
  3 x 1 = 3
  3 x 2 = 6
  3 x 3 = 9
  ...
  3 x 10 = 30
  ```
