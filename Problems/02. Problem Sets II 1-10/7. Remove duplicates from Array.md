


### 7. **Remove Duplicates from Array**
- **Problem**: Write a program to remove duplicate elements from an array of integers.
- **Input**: An array of integers.
- **Output**: The array with duplicate elements removed.

**Example**:
- Input: `[1, 2, 3, 2, 4, 5, 1]`
- Output: `[1, 2, 3, 4, 5]`
